# something

## 简介
本仓库用于保存字幕文件及字幕样式模板

## 索引
- 字幕：[subtitles](subtitles/)
- 旧版字幕：[subtitles/old_versions](subtitles/old_versions)
- 模板：[template](template/)
### 文件命名规则
- 字幕：原作英文名-版本（如有）-制作完毕日期-样式（如有）.扩展名  
  示例：BattleplanArclight-v3-250818-Nixie.ass
- 模板：名称-适用分辨率.扩展名  
  示例：style1-7680x4320.ass
